# Project Overview
This is a responsive personal portfolio website built using React, SCSS, and Framer Motion.  
It showcases my skills, projects, education, and contact links with a modern UI.


# How to Run /view the project
git clone https://github.com/Pranav9555/Resume-ui.git
cd Resume-ui
npm install
npm run dev
